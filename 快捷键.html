以下是一些常用的VSCode快捷键：

  1. 文件操作：
    1. 打开文件：Ctrl + O
    2. 保存文件：Ctrl + S
    3. 关闭文件：Ctrl + F4 或 Ctrl + W
    4. 新建文件：Ctrl + N
    5. 打开文件夹：Ctrl + K Ctrl + O
  2. 编辑操作：
    1. 撤销：Ctrl + Z
    2. 重做：Ctrl + Shift + Z
    3. 复制：Ctrl + C
    4. 粘贴：Ctrl + V
    5. 剪切：Ctrl + X
    6. 删除行：Ctrl + Shift + K
    7. 选择当前行：Ctrl + L
    8. 选择当前单词：Ctrl + D
    9. 跳转到行：Ctrl + G
    10. 查找：Ctrl + F
    11. 替换：Ctrl + H
    12. 直接跳转到下一行：Ctrl + Enter
    13. 在当前行的上方插入一个新的空行，并将光标移动到新行中Ctrl + Shift + Enter
    14. 将光标所在行直接移动到下一行：Alt + Down Arrow
    15. 将当前行向上移动一行，并将光标移动到新行的开始位置Alt + Up Arrow
    16. 注释：Ctrl+/
  3. 导航：
    1. 跳转到文件：Ctrl + P
    2. 跳转到符号：Ctrl + Shift + O
    3. 跳转到定义：F12
    4. 返回：Alt + Left
    5. 前进：Alt + Right
    6. 窗口操作：
    7. 分屏：Ctrl + \
    8. 切换侧边栏可见性：Ctrl + B
    9. 切换全屏：F11
    10. 隐藏活动编辑器：Ctrl + K Z
  4. 终端：
    1. 打开终端：`Ctrl + ``
    2. 新建终端：`Ctrl + Shift + ``
    3. 切换终端：Ctrl + Tab
  5. 调试：
    1. 开始/继续调试：F5
    2. 停止调试：Shift + F5
    3. 步入：F11
    4. 步过：F10
    5. 步出：Shift + F11
  6. 代码格式化：
    1. 格式化文档：Shift + Alt + F
